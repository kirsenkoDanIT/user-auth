# user-auth

[Visit on Heroku](https://auth-user-test.herokuapp.com/)

## Used technologies:
* React
* Redux
* Redux-Saga
* JWT
* Node.js
* Express
* Mongoose
* MongoDB

## Start

### In the root directory

* npm i (concurrently)
* npm run allModules (install server&client modules)
* npm run startAll (start project)
