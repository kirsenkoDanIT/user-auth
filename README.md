# user-auth

[Visit on Heroku](https://user-authorize-app.herokuapp.com/)

## Used technologies:
* React
* Redux
* Saga
* JWT
* Node.js
* Express
* Mongoose
* MongoDB

## Start

### In the root directory

* npm i (concurrently)
* npm run allModules (install server&client modules)
* npm run startAll (start project)
